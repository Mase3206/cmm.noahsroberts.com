Reformat

Reformat is all about data. It has a bunch of functions that just change the format of data it receives. For example, one function takes a list of lists, where each sublist has a complete set of arguments (i.e. [['a=1', 'b=1'], ['a=2', 'b=2']]) and moves them around such that each sublist then has all the values for each argument (i.e. [['a=1', 'a=2'], ['b=1', 'b=2']]). This seems like a novel task, and it is, but it is extremely important for another project I worked on called Handbrake Auto-Transcode. If you're interested in learning more about that project, you can visit the repository at Mase3206/HandBrake-Auto-Transcode or this page on my website.
